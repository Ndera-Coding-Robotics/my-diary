# 🌟 My Diary

A clean, beautiful personal diary app for capturing daily thoughts, tracking habits, and reflecting on meaningful moments.

## 🚀 Project Overview
**My Diary** is a web-based journaling app designed for quick writing, thoughtful reflection, and easy organization.
It includes features for:
- writing daily entries
- choosing writing prompts
- assigning categories and tags
- searching saved entries
- editing and deleting entries
- switching between light and dark themes

## ✨ Key Features
| Feature | Description |
|---|---|
| **Smart prompts** | Predefined writing prompts help you start your journal entry quickly. |
| **Entry editor** | Save new entries or update existing ones with a single click. |
| **Categories & tags** | Organize entries by category and searchable tag keywords. |
| **Local storage** | Entries are stored in the browser so your diary stays available locally. |
| **Search & filter** | Find past entries by text, category, tags, or prompt. |
| **Dark mode** | A toggle button switches between light and dark themes instantly. |

## 🧠 How It Works
1. Open `index.html` in your browser.
2. Type your diary entry into the text area.
3. Optionally select a prompt, category, and tags.
4. Click **Save Entry** to store your thoughts locally.
5. Use the search box to find past entries quickly.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Browser `localStorage`

## 📁 Project Structure
- `index.html` — main diary interface
- `style.css` — styling and theme support
- `script.js` — diary behavior, storage, and entry management

## 💡 Notes
- Entries are saved in browser storage and remain available across sessions on the same machine.
- Word count tracking helps keep each entry under a friendly limit.
- Prompt selection makes journaling easier when inspiration is low.
